# Tool Inventory

The Brightline Studios stack, organized by category. The Tool Mapper uses these to substitute specific vendors into SOP drafts that reference tool *categories* generically.

## Project management

- **ClickUp** — used for client workspaces, internal task tracking, and recurring SOP execution logs

## Communication

- **Slack** — internal team + per-client client-shared channels
- **Loom** — async video updates to clients

## Design + asset creation

- **Figma** — design files for occasional web work and ad creative
- **Canva** — fast social and ad creative production

## Analytics + measurement

- **Google Analytics 4** — web analytics, all clients
- **Looker Studio** — client-facing reporting dashboards

## Email + marketing automation

- **Klaviyo** — e-commerce client email + SMS
- **Customer.io** — B2B SaaS client email (lighter use)

## SEO + search

- **Ahrefs** — primary SEO tool (rank tracker + backlink analyzer + keyword research + site crawler)
- **Google Search Console** — direct query/page performance per client
- **Screaming Frog** — deeper technical crawls when Ahrefs's crawler isn't enough
- **PageSpeed Insights** — Core Web Vitals checks (lab data via web tool, no paid CWV monitoring)

## Paid media

- **Google Ads** — search + Performance Max + shopping for e-commerce clients
- **Meta Ads Manager** — Facebook + Instagram ads
- **Triple Whale** — e-commerce attribution + creative reporting (D2C clients only)

## Social + content distribution

- **Buffer** — social scheduling for the small share of clients we manage organic social for
- *(no formal influencer or community-management tooling)*

## AI + automation

- **Claude (via Claude Code)** — internal development + this SOP framework
- **ChatGPT Plus** — general ad-hoc use
- *(no n8n, Zapier, or other workflow automation in production yet)*

## Storage + documentation

- **Google Drive** — client deliverables and shared docs
- **Notion** — internal wiki + this overlay's edit destination

| backlink-analyzer | Ahrefs |
| keyword-research-tool | Ahrefs |

| web-analytics | Google Analytics 4 |

When the Tool Mapper flags a gap (e.g., log-analyzer, content-optimization-tool), Brightline's typical resolution is: skip the step if the SOP can work without it, or use a free / manual alternative. The team does not buy tools speculatively.
