# Team Structure

## Departments

Brightline has a flat structure. "Departments" below are loose groupings, not formal hierarchies — everyone reports to the founder.

- **Leadership** — Founder (1)
- **Acquisition** — SEO Lead, Paid Media Specialist (2)
- **Retention** — Retention Specialist (1)
- **Client Success** — Account Manager (1)
- **Website** — Designer-Developer (1)

Total headcount: 6.

## Pods / working groups

No formal pods. The team is small enough that ad-hoc collaboration replaces formal pod structure. The Account Manager acts as the cross-functional coordinator when an initiative touches multiple specialists.

## Capacity defaults

For gap-analysis in the Org Mapper:

- A **principal** role can own ~1 concurrent SOP-driven workflow (the founder is fully booked with client strategy; do not assign routine SOP execution)
- A **senior** role can own ~2 concurrent SOP-driven workflows
- A **mid** role can own ~3 concurrent SOP-driven workflows
- A **junior** role would own ~4 (Brightline currently has none; ignore for now)

The Org Mapper should flag any role projected to exceed these capacity defaults across the active SOP set.

## When the Org Mapper flags a gap

Brightline's gap-resolution preference order:

1. **Skip the step** if the SOP can work without it (often the right call for a Lean shop)
2. **Use a contractor / freelancer** — Brightline has working relationships with a copywriter, a designer, and a developer
3. **Defer the SOP** until next hiring cycle (rarely the right call)
4. **Hire** — only if the role would be ≥ 50% utilized across all active SOPs

Note the order. The agent should NOT default to "you need to hire" unless the workload genuinely justifies it.

## Out-of-scope coordination

For service lines Brightline doesn't run in-house (branding, social, agentic-ai, cold email), the agent should:

- Not generate SOPs for those service lines unless explicitly requested
- If requested anyway, mark the SOP as `[OUT-OF-SCOPE]` in the metadata and recommend the user either contract the work out or skip the SOP
