# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its code of conduct. The full canonical text is maintained upstream at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## In short

Everyone interacting with this project — through code, issues, pull requests, discussions, or any other channel — is expected to:

- Be respectful and welcoming
- Assume good faith in others
- Give and accept constructive feedback gracefully
- Take responsibility and apologize when mistakes happen
- Focus on what is best for the community and the project

Maintainers reserve the right to remove, edit, or reject contributions that violate the Contributor Covenant, and to ban contributors whose behavior they deem inappropriate.

## Reporting

To report a concern, contact the project maintainers privately:

- **Email:** security@atlasdigitalusa.com
- **GitHub:** open a *private* security advisory on this repository

All reports are reviewed promptly and confidentially. Reporters will not be retaliated against.

## Scope

This Code of Conduct applies within all project spaces (this repository, its issues, pull requests, and any official communication channels), and also when an individual is representing the project in public spaces.
